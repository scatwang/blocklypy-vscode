# Connect to Your LEGO Device

BlocklyPy Commander supports multiple LEGO devices running Pybricks firmware:

- **SPIKE Prime** / **SPIKE Essential** hubs
- **MINDSTORMS Robot Inventor** hub
- **Powered Up / Technic** hubs
- **EV3** brick (with Pybricks firmware)

or standard LEGO firmware (HubOS v3):

- **SPIKE Prime** / **SPIKE Essential** hubs

## Connection Methods

### Bluetooth Connection (Recommended)

1. Turn on your LEGO hub
2. Click the **Connect Device** button in the BlocklyPy Commander sidebar
3. Select your device from the list
4. Wait for the connection to establish

### USB Connection

Some devices support USB connection for faster data transfer:

1. Connect your hub via USB cable
2. Click **Connect Device**
3. Select the USB device

## Tips

- Make sure your device is powered on and in range
- If using Bluetooth, ensure it's enabled on your computer
- The extension will remember your last connected device for quick reconnection

Once connected, you'll see the device name and status in the status bar.
